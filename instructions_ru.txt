=== ИНСТРУКЦИЯ ПО ЗАПУСКУ СКРИПТА PHAROS ===

✅ 1. Необходимо:
- Node.js версии 16+ (Скачать: https://nodejs.org)
- Файлы: priv.txt, proxies.txt
- Проверь установку: node -v && npm -v

✅ 2. Установка зависимостей:
$ npm install

✅ 3. Файлы:
- priv.txt (приватные ключи, по одному в строку)
- proxies.txt (список HTTP/S прокси)
- папка logs/ создается автоматически

✅ 4. Запуск скрипта:
$ node main.js

Выбери опцию из меню:
1 - Запуск с фаусетом
2 - Запуск без фаусета
3 - Полная ликвидность
4 - Полные свопы и конвертации
5 - Только фаусет

Затем введи:
- Потоки (1–100)
- Итерации (например, 10–25)

✅ 5. Логи:
- logs/success_log/
- logs/failed_log/

✅ 6. Примечания:
- Формат прокси: http://логин:пароль@ip:порт
- Прогресс будет отображаться в консоли
